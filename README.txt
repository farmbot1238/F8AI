📚 LMJO 09 - المساعد الذكي للمواد الوزارية
=============================================

🔧 طريقة التشغيل:
-----------------
1. قم بتحميل جميع الملفات الثلاثة في مجلد واحد
2. افتح ملف index.html في المتصفح
3. اضغط على زر "إعدادات API" وأدخل مفتاح API الخاص بك
4. اختر نوع API (OpenAI أو Gemini)
5. ابدأ بطرح الأسئلة!

🔑 الحصول على مفتاح API:
------------------------
• OpenAI: https://platform.openai.com/api-keys
• Google Gemini: https://makersuite.google.com/app/apikey

📖 المواد المدعومة:
-------------------
✓ التاريخ (الفصل الأول والثاني)
✓ الدين الإسلامي (الفصل الأول والثاني)
✓ اللغة الإنجليزية (جميع المستويات)
✓ اللغة العربية (النحو والأدب)

💡 نصائح:
---------
- يمكنك اختيار مادة محددة للتركيز عليها
- الأسئلة خارج المواد المدعومة سيتم الاعتذار عنها
- يتم حفظ مفتاح API محلياً في المتصفح

🎯 مميزات LMJO 09:
------------------
- واجهة عربية بالكامل
- دعم الفصلين الأول والثاني
- إجابات دقيقة ومركزة
- حفظ الإعدادات تلقائياً
- تصميم عصري وجذاب

📞 للدعم:
---------
في حال وجود أي مشكلة، تأكد من:
1. صحة مفتاح API
2. الاتصال بالإنترنت
3. تحديث المتصفح
